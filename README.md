# ElevateX 🚀

ElevateX is a cloud infrastructure optimization platform conceptualized for an Innovation & Entrepreneurship course project. 

> [!WARNING]
> **This repository is a Frontend Demo only.** It contains the UI/UX mockups, marketing pages, and dashboard interfaces. It does not currently contain the backend autonomous engine for interacting with AWS/GCP/Azure APIs.

## Features
- 🌙 **Premium "Night Obsidian" UI**: Built with Tailwind CSS and Framer Motion for a sleek, glassmorphic dark mode aesthetic.
- 📊 **Dashboard Mockups**: Interactive UI simulating real-time cloud resource monitoring and idle instance tracking.
- 💰 **ROI Calculator**: Interactive slider to demonstrate potential cost savings.
- 🚀 **Next.js App Router**: Modern React architecture.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the platform.

## Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS
- Framer Motion
- Lucide Icons
